# SkillCraft Task 04 - Traffic Accident Data Analysis

## Objective
Analyze traffic accident data to identify patterns related to weather conditions, accident severity, and time of day. Visualize accident hotspots and contributing factors.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Folium

## Dataset
US Accidents March 2023 Dataset

## Features
- Data Cleaning
- Weather Condition Analysis
- Time-based Accident Analysis
- Accident Severity Visualization
- Accident Hotspot Mapping

## Output Files
- accidents_by_weather.png
- accidents_by_time.png
- accident_severity.png
- hotspot_map.html
- summary.txt
